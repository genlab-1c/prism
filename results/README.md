# results/

Сюда складываются результаты прогонов:

- `experiment_*.json` — сырые ответы моделей (пишет `prism generate`);
- `auto/` — автоматические оценки Уровня 1 (пишет `prism score`). Не версионируются:
  это воспроизводимый дериват из ответов моделей + правил метрики (`auto/` в `.gitignore`).

Пусто до первого открытого прогона. Мартовские результаты удалены намеренно — цифры будут
перегенерированы на актуальных моделях. Исходный мартовский эксперимент сохранён в
замороженных репозиториях (`core-bench` / `prism-concept`) для воспроизводимости статьи.
